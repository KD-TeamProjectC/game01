1. ゲーム概要

ゲーム名（仮）
ぬるぽ判定・反射神経テスト

ジャンル
反射神経テスト＋判別ゲーム

目的
表示された文字列が

クリック対象か否かを瞬時に判断し

正しい場合のみクリックし、反応時間を競う

2. 基本ルール
表示される文字の種類
分類	説明	クリック
正解語	完全一致で「ぬるぽ」	○
擬似正解語	「ぬるぽ」を含む／同義的	○
誤答語	似ているが無関係	×
例

正解語

ぬるぽ

擬似正解語（クリック対象）

NullPointerException

いろはにほへとちりぬるぽ

誤答語（クリックすると失敗）

ぬろぽ

ぬかぽコォ

3. ゲーム進行

スタートボタン押下

文字列を1つ表示

プレイヤーが判断してクリック or 何もしない

結果表示

リトライ可能

4. 判定仕様
正解条件

表示文字が

正解語

擬似正解語

失敗条件

設計書
1. 状態設計（State）
IDLE      : 開始前
WAITING   : ランダム待機中
SHOWING   : 文字表示中
RESULT    : 結果表示

2. データ設計
文字列データ構造
const words = [
  { text: "ぬるぽ", isCorrect: true },
  { text: "NullPointerException", isCorrect: true },
  { text: "いろはにほへとちりぬるぽ", isCorrect: true },
  { text: "ぬろぽ", isCorrect: false },
  { text: "ぬかぽコォ", isCorrect: false }
];


※ 判定は 完全にフラグ依存
→ 文字解析不要で実装が軽い

4. 処理フロー
スタート処理

state = WAITING

ランダム時間で setTimeout

ランダムで words から1件選択

表示＋ showTime 記録

state = SHOWING

クリック処理
IF state === WAITING
 → フライング失敗

IF state === SHOWING
 → currentWord.isCorrect
    true  → 成功（反応時間計測）
    false → 誤クリック失敗
